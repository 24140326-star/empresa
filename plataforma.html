Accesibilidad para Todos

En VitalMove creemos que la tecnología debe ser accesible para todas las personas, sin importar sus capacidades. Nuestra plataforma está diseñada para ser utilizada de manera sencilla y efectiva, con herramientas de asistencia y navegación optimizadas.

Implementamos opciones de accesibilidad como ajuste de tamaño de fuente, colores de alto contraste y texto alternativo para imágenes, para que todas las personas puedan disfrutar de nuestros servicios.

Diseño Inclusivo

El diseño inclusivo es una prioridad para nosotros. Hemos creado una plataforma que se adapta a diferentes dispositivos y tamaños de pantalla. La interfaz es simple y clara, asegurando que sea fácil de navegar para todas las personas, incluidas aquellas con discapacidades visuales, motoras o cognitivas.

Algunas características incluyen:

  • Ajuste de contraste alto y opciones de color.
  • Textos grandes y legibles.
  • Teclado virtual y compatibilidad con lectores de pantalla.
  • Fácil navegación con teclas de acceso rápido.

Compatibilidad con Todos los Dispositivos

Sabemos que cada usuario tiene diferentes necesidades y dispositivos. Por eso, nuestra plataforma es completamente compatible con una amplia variedad de dispositivos, incluyendo:

  • Ordenadores de escritorio y portátiles.
  • Tabletas y teléfonos móviles.
  • Dispositivos de asistencia como lectores de pantalla y teclados alternativos.

Además, nuestra plataforma funciona sin problemas en diferentes navegadores web y sistemas operativos.
